## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client approve → /apply-patch | the one consequential write; editor-gated + CSRF + a real-diff human approval. |
| proposal-time revision → apply-time revision | a concurrent edit may move the revision between propose and approve — must block, not overwrite. |
| agent output → file | the ONLY path is the human-approved /apply-patch → pages.Save; no tool-driven write exists. |
